# Unusual Museums
An API with Python and Flask


## Installation

### Install dependencies
`pip3 install -r requirements.txt`

### Run locally
`python3 app.py`

## Endpoints
`/api`<br>
This will return an list of all the museums

`/api/:id`<br>
Will return a museum with the ID specified
